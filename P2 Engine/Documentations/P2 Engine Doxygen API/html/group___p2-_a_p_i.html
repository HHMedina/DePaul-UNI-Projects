Modules

 GameObject
 All game entities will be derived from the GameObject class.
 
 Sound
 This module teaches the user how to use sound in the P2 Engine.
 
 AssetManager
 The AssetManager is used to allow the player to load models and texture into the game, while also allowing them to get a reference to that loaded resource.
 
 GraphicsObjectMaker
 The GraphicsObjectMaker is pretty much a factory that will create and return different types of Graphics Object. The GraphicsObjectMaker returns a new graphics object as well as offering more functionality than just creating a new GraphicsObject.
 
 TimeManager
 The TimeManager is used to allow the player to get the frame time and to get the time since the start of the game.
 
 Scenes
 A scene is a base class from which all scenes should derive from.
