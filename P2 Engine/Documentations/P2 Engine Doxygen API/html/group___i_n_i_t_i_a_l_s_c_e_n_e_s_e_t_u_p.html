Setting the initial scene
P2 API » Scenes
Collaboration diagram for Setting the initial scene:

Every game you create must have at least one scene, and must have a starting scene, or else the P2Engine will have no idea where to start. In order to set the initial scene, the user has to use the function SceneManager::SetStartScene() in the P2EngineContent.cpp's LoadGameContent() which is located in the User Files folder.

Below is an example of setting the initial scene, where the scene is the same scene from the "Creating a Level" section in the Scene module of the API.
void Game::LoadGameContent(){ // this is located in the P2EngineContent.cpp in the User Files
//make sure to load all the assets you will be using through out all of the levels.
AssetManager::LoadModel("space_frigate.fbx","SpaceShip");
AssetManager::LoadTexture("grid.tga","grid");
SceneManager::SetStartScene(new TutorialLevel1()); //sets the first scene to the TutorialLevel1
}
