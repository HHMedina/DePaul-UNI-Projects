Detailed Description

These are the events that can be performed by a scene.

Note
All derived classes from scene will inherit these functions.

void SceneManager::ChangeScene ( Scene *  nScene)
static

Function used to change scenes.This method is used to change to the next scene, by passing in what scene to go to.

Parameters
nScenenScene is pointer to a Scene that will be changed into.
Example:
The following code changes level when a Player (class) collides with a door.
void Player::Collision(Door* other){
SceneManager::ChangeScene( new HospitalInterior() );
}

template<class C1 , class C2 >
void Scene::SetCollisionPair ( )
inlineprotected

Sets the collision pairs for a scene. 

This method will allow the user to specify which pair wise collisions should be considered by the collision system.
This method should be called in the SetUp() method of a derived Scene class.

.

Example:
The Following code shows setting a collision pair between WaterBalloon GameObjects and Clown GameObjects.
void CarnivalLevel::SetUp(){
Clown* bobo = New Clown();
Clown* sadSally = new Clown();
BadKid* dennis = new BadKid();
WaterBalloon* wBalloon = new WaterBalloon();
SetCollisionPair<WaterBalloon,Clown>();
}
Note
A user should not set collision pair between WaterBalloon type and the Clown Type and between the Clown Type and WaterBalloon Type. Only one is needed and should be done.
template<class C >
void Scene::SetCollisionSelf ( )
inlineprotected

Sets the considerations for Self-class collisions. 

This method will allow the user to specify that a Collidable of Class C can collide with other Collidables of
the same class.

.

Example:
The Following code shows setting a collision pair between Marble GameObjects.
void MarblesGameLevel1::SetUp(){
Marble* marble1 = new Marble();
Marble* marble2 = new Marble();
SetCollisionSelf<Marble>(); // make MArbles able to collide with each other.
}

An example of setting a scene is seen below.
void Game::LoadGameContent(){
// Model loading ---------------------------------------
//Model loading omitted.
// Texture Loading --------------------------------
//texture loading omitted.
SceneManager::SetStartScene(new TutorialLevel1() );
}
virtual void Scene::SetUp ( )
protectedpure virtual

Function used to initialize a scene.This method is used to initialize a scene, so any GameObjects, or scoring, should be initialized here. This method is called when the game starts and when a scene changes, calling the SetUp() of the new scene.

Example:
The Following code the SetUp() for a scene which is supposed to have a desert terrain and a tank (controlled by the player)
void DesertLevel::SetUp(){
Tank* player = new Tank();
Terrain* desert = new DesertTerrain();
SetCollisionPair<Bullet,Tank>(); // in this level, there will be multiple tanks who shoot each other with bullets
}
Note
Every derived level is forced to reimplement the SetUp() method.
