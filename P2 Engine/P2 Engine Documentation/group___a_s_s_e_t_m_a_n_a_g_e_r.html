Detailed Description

The AssetManager is used to allow the player to load models and texture into the game, while also allowing them to get a reference to that loaded resource.

Function Documentation

Model * AssetManager::GetModel ( const char *const  assetName)
static

This method gets a pointer to the wanted Model asset.

This method returns a pointer to a Model which is retrived via assetname. If this method does not find the assset, it will fail. If it does, check to see if you have loaded the asset correctly.

Parameters
assetNameassetName is the name of the Model the user wants a reference to.
Example:

The following is an example of a user making a graphicsObject out of a model and a texture to visually represent a Ship GameObject.

Ship::Ship(){
SetGraphicsObject( GraphicsObjectMaker::CreateGraphicsObjectFlatTexture(AssetManager::GetModel("SpaceShipModel"),AssetManager::GetTexture("ShipTexture")));
}
Texture *const AssetManager::GetTexture ( const char *const  assetName)
static

This method gets a pointer to the wanted Texture asset.

This method returns a pointer to a Texture which is retrived via asset name. If this method does not find the assset, it will use a purple texture instead.

Parameters
assetNameassetName is the name of the Texture the user wants a reference to.
Note
For an examaple, please see GetModel() , in which the user uses GetModel() and GetTexture() to set the visual representation of a Ship GameObject.
void AssetManager::LoadModel ( const char *const  fileName,
const char *const  assetName 
)
static

This method loads a file into the game.

Note
an asset should be loaded by using the P2EngineContent.cpp's LoadGameContent() located in the User Files folder. This method loads a model located in the "assets" folder and then loads it into the game's asset database as a Model.
Parameters
fileNamefileName is the name (with extension) of the file to be loaded.
assetNameassetName is the name the user sets to the asset, which will be used when loading assets.
Example:

The following is an example of a user loading a ship model and texture, as well as a turtle model and texture.

void Game::LoadGameContent(){ //located in P2EngineContent.cpp in the User Files Folder.
// Model loading ---------------------------------------
// Load external models -------------------------------
AssetManager::LoadModel("space_frigate.fbx","SpaceShipModel");
AssetManager::LoadModel("turtle.fbx","TurtleModel");
// Texture Loading --------------------------------
AssetManager::LoadTexture("space_frigate.tga","ShipTexture");
AssetManager::LoadTexture("turtle.tga","TurtleTexture");
SceneManager::SetStartScene(new Level1()); // set the starting scene of the game. Important! See the Scene Module.
}
