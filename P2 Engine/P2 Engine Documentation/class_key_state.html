bool KeyState::IsKeyStateEmpty ( )
private

Checks if the KeyState is empty.

This function checks if the keystate is empty by checking if the keyUpList and keyDownList are both empty. The function returns true if both lists are empty. If the lists are not both empty, then it will return false.

void KeyState::Update ( )
private

Updates the state of the key.

This method checks if the Key for this KeyState Object has been pressed by using CheckKeyDown() , checks if the key has been released by using CheckKeyUp(), and then records the current state of the key as the previous state.
