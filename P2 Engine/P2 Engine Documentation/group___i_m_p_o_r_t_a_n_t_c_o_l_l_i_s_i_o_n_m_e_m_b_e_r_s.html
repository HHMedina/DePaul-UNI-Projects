These are important members related to Collisions.

Note
A GameObject is Derived from Terminable, Collidable, Updatable, Inputable, Drawable, and Alarmable, and will therefore absorb all of its methods.

Model* Collidable::collisionModel
protected

This is the Model which is the repsentation of the collision mesh.

This model will be used to calculate if another collidable has collided with this collidable

Matrix Collidable::colWorld
protected

This is the world Matrix of the collisonModel which holds its position, scale, and rotation.

This Matrix will be used to set the rotation,location, and scale of the Collision Model.
