Detailed Description

The TimeManager is used to allow the player to get the frame time and to get the time since the start of the game.

Function Documentation

float TimeManager::GetFrameTime ( )
static

This function returns the time for a single frame.

The following prints out the frametime as part of a Ship's Update().
#include "Ship.h"
#include <iostream>
#include "TimeManager.h"
Ship::Update(){
cout<< TimeManager::GetFrameTime();
}
